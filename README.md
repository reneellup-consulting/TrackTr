# Tracktr - A Fleet Tracking Solution

Tracktr is a proprietary GPS tracking system based on Traccar source code. We modified it and added more features and 
a more modern Web UI, mobile UI for Android and IOS.

This repository contains Java-based back-end service, React JS as its front-end and React Native for mobile 
applications. The same as Traccar, it supports more than 170 GPS protocols and more than 1500 models of GPS tracking 
devices.

Tracktr can integrate with any significant SQL database system.

## Features

Available features include:

- Real-time GPS tracking
- Driver behavior monitoring
- Detailed and summary reports
- Geofencing functionality
- Alarms and notifications
- Account and device management
- Email and SMS support
